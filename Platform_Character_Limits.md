# Social Media Character Limits — Quick Reference
**Fincare / aifincare.com**
*Always check this before writing any post*

---

## LINKEDIN

| Element | Limit |
|---|---|
| Personal profile post | **3,000 characters** |
| Company page post | **700 characters** |
| Preview before "see more" | **~210 characters** (~3 lines) |
| Hashtags (recommended) | **3–5 max** |
| First comment | **1,250 characters** |
| Article | No limit |

**Rules:**
- Always post the link in the **first comment**, not the post body (links cut reach ~40%)
- Company page = 700 chars max. Personal profile = 3,000 max. Always have both versions ready.
- First 3 lines are everything — write the hook before the "see more" cut

---

## INSTAGRAM

| Element | Limit |
|---|---|
| Caption | **2,200 characters** |
| Preview before "more" | **~125 characters** |
| Hashtags | **30 max** (best practice: 10–15) |
| Slide text (carousel) | No strict limit — keep under **100 chars/slide** |
| Story text | **No limit** (but keep it visual — short) |
| Bio | **150 characters** |
| Comment | **1,000 characters** |

**Rules:**
- First 125 characters of caption = your hook. Make it count.
- Hashtags go at the end of the caption or in the first comment
- Carousel slides: less is more — one idea per slide, short text wins

---

## TIKTOK

| Element | Limit |
|---|---|
| Caption | **2,200 characters** |
| Caption preview | **~100 characters** |
| Hashtags | No strict limit (best practice: 5–10) |
| On-screen text overlay | No strict limit — keep to **1–2 lines/frame** |
| Bio | **80 characters** |

**Rules:**
- Caption preview (~100 chars) must hook immediately — people decide to read more in 1 second
- On-screen text: maximum 2 short lines per frame. Any more and viewers look away.
- Bio: one punchy line + "🔗 aifincare.com" — that's it
